📚 数据结构与算法说明

🏰 场景描述

模拟银行多窗口服务：客户到达后进入等待队列，空闲窗口自动叫号，服务完成后记录压入历史栈，支持撤销回退。

📚 队列 Queue（先进先出 FIFO）

  • 普通客户到达 → queue.push(c) 加入队尾
  • VIP客户到达 → queue.unshift(c) 插入队头，优先叫号
  • 窗口叫号 → queue.shift() 从队头取出

📋 栈 Stack（后进先出 LIFO）

  • 服务完成 → history.push(record) 压入栈顶
  • 撤销操作 → history.pop() 弹出栈顶，客户重回队头
  • 栈顶始终是"最近完成的服务"

核心伪代码：

// 入队 if (vip) queue.unshift(customer) // 队头插入 else queue.push(customer) // 队尾追加 // 窗口叫号（每步执行） for each idle window: if queue.length > 0: c = queue.shift() // dequeue window.serve(c, duration) // 服务完成 history.push({ customer, window, clock }) // Stack push // 撤销 rec = history.pop() // Stack pop queue.unshift(rec.customer) // 重新入队头
