📚 算法原理 & 数据结构

核心数据结构是栈(Stack)，即后进先出(LIFO)。

  • 工厂队列：按给定顺序取出 = Queue
  • 临时货架：只能从顶部存取 = Stack
  • 装填目标：严格按 1,2,...N 顺序

伪代码：

currentNeed = 1
while currentNeed <= N:
  if 栈顶 == currentNeed: 弹出装填, currentNeed++
  else if 工厂有货: 取一箱 box
    if box == currentNeed: 装填, currentNeed++
    else if 栈满: 失败!
    else: 压入栈
  else: 失败!

💡 关键结论

  1. 顺序发货 1,2,...N 无需货架，直接成功。
  2. 逆序发货需至少 N-1 容量的货架。
  3. 货架容量越大，成功概率越高。
  4. 失败原因：所需颜色被压在栈中间 或 货架已满。
